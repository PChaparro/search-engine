## 🛠️ Install and run

### Install packages

```bash
pnpm i 
```

or

```bash
npm i 
```

### Run: 

**Note:** Before start front-end side you should start the back-end side. 

```bash
pnpm run dev
```

or

```bash
npm run dev
```
